02 / concept

Why a sawtooth

The mark encodes the algorithm; the algorithm encodes the product.

AIMD — additive-increase / multiplicative-decrease — is the congestion-control discipline TCP has run for forty years and what this proxy runs on every bearer. The live ceiling ramps additively while upstream stays healthy and drops multiplicatively the instant Anthropic returns 429 or 503.

The sawtooth is the picture you would draw if you graphed that ceiling over time. Slow climbs. Sharp cuts. Each cut leaves a pushback marker at the corner. Multiply by N bearers and the system breathes.

That is why the mark is a sawtooth and not an abstract glyph. A generic "AI proxy" wordmark would carry no information. This one tells you what the binary is doing the moment you see it.

05 / voice

Operator-first prose

Facts and numbers, never adjectives. Lowercase. Mono. The proxy reports what it did; the human decides how to feel.

06 / guardrails

Refuse on sight

The brand is defined as much by what it rejects. These are hard rules — every render gates against them.

No gradients, no neon glow, no glassmorphism

The mark sits on solid Catppuccin surfaces only. A blurred translucent panel is a tell the design has nothing to say underneath it.

No rocket / sparkle / shield iconography

The AI-starter-pack icon set is banned. The product is congestion control, not magic.

No raw hex in components

Catppuccin tokens only — every colour gets a name and a role. Raw hex outside this board's token table is a bug.

The mark stays a single-weight line

No 3D, no extrusion, no app-store render, no fill, no rotation. The sawtooth is the data; embellishment is noise.

Decorative colour is forbidden

Teal = brand. Green / peach / red = health state. Blue = link. Anything else asks itself: what does this mean?

No exclamations, no adjectives, no hype

"1 of 3 bearers throttled" beats "uh-oh, rate limited!". The operator reads logs, not announcements.
